# متجر جاهز (Static Store)

ملفات المتجر موجودة في نفس الفولدر وتشتغل مباشرة في VS Code.

## تشغيل سريع

1) افتح `index.html` في المتصفح.

2) الأفضل (عشان كل حاجة تشتغل بدون قيود): شغّل سيرفر محلي من نفس المجلد:

```powershell
python -m http.server 5173
```

وبعدين افتح:
`http://localhost:5173/`

## تعديل المنتجات والسعر والصور

- أسهل طريقة: افتح `admin.html` وعدّل المنتجات ثم اضغط **حفظ المنتجات**.
  - باسورد لوحة التحكم الافتراضي: `312007`، وتقدر تغيره من الإعدادات.
  - على GitHub Pages القفل ده يحمي الواجهة فقط، والأمان الكامل يحتاج backend.
  - التعديلات بتتخزن في المتصفح (LocalStorage).
  - لو عايز تنقل التعديلات لملف، استخدم **تصدير JS** واستبدل محتوى `data/store-data.js`.

- تعديل يدوي (مباشرة من VS Code):
  - المنتجات: `data/store-data.js`
  - الإعدادات (اسم المتجر/العملة/الشحن/واتساب): `data/store-data.js`

## واتساب وحفظ السلة

- لينك واتساب موجود في `STORE_CONFIG.whatsappLink`، وبعد تأكيد الطلب الموقع يفتح واتساب تلقائيًا مع زر احتياطي.
- زر الحساب في الهيدر يحفظ السلة حسب الاسم ورقم الموبايل على نفس المتصفح باستخدام LocalStorage، وده يشتغل على GitHub Pages بدون سيرفر.

## إضافة صورك

ضع صور المنتجات داخل `assets/store/` ثم حدّث مسارات `images` داخل `data/store-data.js` أو من لوحة التحكم.

- في لوحة التحكم تقدر تكتب كل صورة في سطر مستقل أو تفصل بينها بفواصل.
- لو كتبت اسم الصورة فقط مثل `tshirt-front.jpg` هيتم اعتبارها تلقائيًا داخل `assets/store/`.
- الزوم يستخدم نفس ملف الصورة الأصلي بدون ضغط أو تصغير، فحط صور واضحة WebP/JPG/PNG عشان التفاصيل تظهر قوية.
